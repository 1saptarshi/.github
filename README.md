 # 🌟 GitHub Sponsorship Configuration

This `.github` repository contains shared community health files and funding configuration for the `@1saptarshi` GitHub account.

## 🤝 Sponsor Button Enabled

The `FUNDING.yml` file located in this repository enables the **Sponsor** button on:

- The `@1saptarshi` user profile
- All public repositories under this account

## 💸 How to Support

You can support this work through one of the listed platforms. Simply click the **Sponsor** button on the profile or any repo to see available options.

## 📁 Files in This Repository

- `.github/FUNDING.yml`: Configuration file listing supported funding platforms.
- `README.md`: This file, explaining the purpose of the repo.
